# Python File Manager

A simple command-line file manager built using Python that performs basic file operations.

## Features

- Create a file
- Read file content
- Update file (rename, overwrite, append)
- Delete a file
- List all files and folders in the directory

## Technologies Used

- Python
- pathlib module
- os module

## How to Run

1. Clone the repository

git clone https://github.com/yourusername/python-file-manager.git

2. Run the program

python main.py

## Example

Select option from menu:

1 → Create File  
2 → Read File  
3 → Update File  
4 → Delete File  
5 → Exit

## Learning Purpose

This project demonstrates file handling and CLI-based CRUD operations in Python.
